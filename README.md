# ☕ Java Studies Repository

This repository is a **continuous self-taught study project** focused on learning and improving Java programming skills.

Although part of the content is based on a Udemy course, this repository is **not limited to a single course**. It is intended to grow over time with new concepts, exercises, and experiments as I deepen my understanding of Java and software development.

---

## 📚 Purpose

- Practice Java fundamentals and core concepts  
- Reinforce logic and problem-solving skills  
- Explore Object-Oriented Programming (OOP)  
- Build a solid foundation for more advanced topics  
- Serve as a long-term learning archive  

---

## 🧠 Topics Covered

### Java Basics (`JavaLogic`)
- Data input and output  
- Arithmetic operations  
- Logical and comparative expressions  
- Conditional structures (`if`, `switch`)  
- Loops (`for`, `while`)  
- Type conversion  
- String manipulation  

### Object-Oriented Programming (`ObjectOrientedProgramming`)
- Sequential structures  
- Basic class structure  
- Exercises focused on:
  - Encapsulation  
  - Logic building  
  - Real-world problem simulation
